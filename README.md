# Git Workshop

A collection of recipe files for practicing Git workflows and collaboration.

## Description

This repository is designed for educational purposes to help users learn Git version control through hands-on practice. It contains a collection of recipe markdown files (spaghetti, caprese salad, chocolate mug cake) that serve as sample content for practicing Git operations like branching, merging, and resolving conflicts.

## Features

- Simple markdown recipe files for easy editing and merging exercises
- Pre-configured `.gitignore` for common development files
- Lightweight structure ideal for Git learning exercises
- Multiple recipe files to practice collaborative workflows

## Technologies Used

- Markdown
- Git

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/gitworkshop.git

# Navigate to project directory
cd gitworkshop
```

## Usage

```bash
# View the recipes
cat spaghetti.md
cat caprese-salad.md
cat chocolate-mug-cake.md

# Practice Git workflows
git checkout -b your-branch-name
# Make changes to recipe files
git add .
git commit -m "Your commit message"
git push origin your-branch-name
```

Use this repository to practice:
- Creating and switching branches
- Making commits
- Merging branches
- Resolving merge conflicts
- Collaborative workflows with pull requests
